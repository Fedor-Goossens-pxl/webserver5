Technische skills

Technische Ontwikkeling

Netwerkbeheer & Datacommunicatie (OLR 1, 2, 7)

Tijdens mijn stage bij B1TS heb ik een sterke groei doorgemaakt in netwerkbeheer. Ik leerde werken met WatchGuard-firewalls, VLAN-configuraties, trunkpoorten, UniFi-switches en access points. Ik kreeg inzicht in netwerksegmentatie, DHCP-services, routing en troubleshooting van complexe netwerkproblemen.

WatchGuard VLAN UniFi DHCP Routing Troubleshooting

Servers, Clients & Virtualisatie (OLR 2, 3, 5)

Ik installeerde en configureerde Windows- en Linux-servers, beheerde updates, installeerde drivers, testte hardware en werkte met tools zoals Windows Admin Center, Azure, TeamViewer en iLO. Ook leerde ik werken met Veeam Backup & Replication en Pulseway voor monitoring en automatisatie.

Azure Windows Admin Center Veeam Pulseway TeamViewer iLO

Automatisatie & Scripting (OLR 6)

Ik ontwikkelde automatisatiescripts binnen Pulseway om software automatisch te installeren. Ik leerde hoe workflows worden opgebouwd, getest en uitgerold binnen een professionele omgeving.

PowerShell Pulseway Scripting Podman Ubuntu Server
